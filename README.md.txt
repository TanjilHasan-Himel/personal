# Portfolio Website

#### Video Demo: https://youtu.be/jjIEigZqCpY

#### Description:
My project is a portfolio website that showcases my personal information with a minimalistic design. The website features a matrix effect themed with Batman colors. It includes a section about my education, and a button to view and download my resume. Additionally, the website has social media links represented as logos for LinkedIn, GitHub, Facebook, and Gmail. By clicking these logos, users can visit my profiles directly.

Key Features:
- Minimalistic design with a Batman color theme
- Matrix effect for a unique look
- Resume download button
- Social media links (LinkedIn, GitHub, Facebook, Gmail)
